# Citation

If you use ULF-Synth in your work, please cite the arXiv preprint:

```bibtex
@misc{musah2026ulfsynth,
  title        = {ULF-Synth: Physics-Guided Ultra-Low-Field MRI
                  Enhancement for Pediatric Neuroimaging},
  author       = {Toufiq Musah and Salvatore Calcagno and
                  Federica Proietto Salanitri and Xiaomeng Li and
                  Maruf Adewole and Marawan Elbatel},
  year         = {2026},
  eprint       = {2605.24625},
  archivePrefix = {arXiv},
  url          = {https://arxiv.org/abs/2605.24625}
}
```

The citation notice is printed automatically on the first call to any
simulation or enhancement function.
